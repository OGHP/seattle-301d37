We use the "hidden checkbox trick" to toggle the state of the hamburger menu. The menu itself is floated right and the checkbox is hidden. We cover the menu "icon" with a label tag, so that when you click the label, it toggles the checkbox. Then, we can easily style elements adjacent to or near the checked box ...

In desktop mode, we basically hide the burger div, and unfloat everything so that it lays out cleanly.
